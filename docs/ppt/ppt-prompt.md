Reusable Prompt Template

请基于以下文档目录生成两份 PPT（用 python-pptx 脚本生成）：

📂 文档源： docs/dev-plan/

---

### PPT 1：产品概览（非技术版）
- 面向非技术用户（投资人/合作方/普通用户）
- 用通俗语言解释产品是什么、解决什么问题、怎么用
- 不出现技术术语（如 WebSocket、JWT、ORM 等）
- 9-12 slides

### PPT 2：产品视觉（图示版）
- 以图示为主：架构流程图、用户旅程、UI 概念图、功能矩阵
- 用几何图形（矩形、圆形、连线、箭头）在 PPT 内直接绘制
- 10-14 slides

**排版风格：**
- 参考 Typewolf（字体层次）、siteInspire（专业布局）、Siiimple（极简）
- 大标题突出，正文克制，留白充足
- 标题与正文字号比至少 2:1
- 每页信息密度低——宁可多一页，不要挤一页

**绝对禁止：**
- 不用 emoji
- 不用渐变填充
- 不用 WordArt/艺术字
- 不用剪贴画或外部图片

**允许的视觉元素：**
- 细线（1pt）作为分隔和装饰
- 薄边框圆形编号（如 ① ② ③ 用 Oval + 数字绘制）
- 纯色矩形卡片
- 几何图形组合成的流程图/架构图
- 小圆点作为列表符号

### 输出要求
- 生成一个 Python 脚本，运行后输出两个 .pptx 文件
- 宽屏 16:9
- 中文为主
- 脚本运行完后删除脚本本身
